# Análisis de Preferencia: Adopción de Herramientas LLM y Percepción de Confianza

### Criterios:
1. **Herramienta LLM Preferida:** Identificación del modelo de lenguaje de mayor uso entre los encuestados, centrándose en tres principalmente:
    * **ChatGPT**
    * **Claude**
    * **Gemini**

2. **Nivel de Confianza:** Escala cualitativa que mide la fiabilidad percibida por el usuario sobre las respuestas del modelo:
    * **Alto:** Mayor nivel de confianza.
    * **Medio:** Nivel de confianza neutro o condicionado.
    * **Bajo:** Nivel de desconfianza o verificación constante.

3. **Cantidad de Usuarios:** Valor numérico que determina la longitud de las barras radiales, representando el volumen total de usuarios por herramienta y su respectiva segmentación.

### Justificación de los criterios
Se seleccionó el **Radial Bar Chart** debido a su capacidad para presentar la información de 
manera concéntrica, donde cada herramienta se representa como una barra que crece desde el centro hacia la periferia. 
Esta disposición permite comparar visualmente la popularidad de cada LLM mientras se integra una segmentación por colores 
para el nivel de confianza. 

### Conclusiones
* **Correlación Popularidad-Confianza:** Se analiza si las herramientas más utilizadas son también las que proyectan niveles de confianza "Mucho" o "Bastante", o si por el contrario, existe una adopción masiva a pesar de una confianza moderada.
* **Segmentación de Fiabilidad:** La visualización revela la distribución de seguridad del usuario; por ejemplo, permite detectar si alguna herramienta específica domina el segmento de "Confianza Extrema" frente a sus competidores.
