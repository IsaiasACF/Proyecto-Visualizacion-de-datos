# Fuentes de datos

Este directorio reúne las fuentes utilizadas en las visualizaciones del proyecto sobre LLMs.

## 1. Chatbot Arena Human Preference 55K
- **Origen:** Hugging Face Datasets (`lmarena-ai/arena-human-preference-55k`)
- **Descripción:** Dataset de comparaciones entre pares de modelos LLM, con prompts, respuestas y resultado de preferencia humana. Se utilizó para analizar enfrentamientos directos y construir el gráfico Sankey de victorias entre modelos.

## 2. Artificial Analysis
- **Origen:** Artificial Analysis
- **Descripción:** Fuente de métricas comparativas de modelos LLM, incluyendo rendimiento en benchmarks y costos de uso. Se utilizó para recopilar valores de desempeño y costo de inferencia en el gráfico de coordenadas paralelas.

## 3. Open LLM Leaderboard
- **Origen:** Hugging Face Spaces (`open-llm-leaderboard`)
- **Descripción:** Ranking abierto de modelos LLM con resultados en benchmarks estandarizados. Se utilizó como apoyo para complementar y contrastar métricas de rendimiento técnico en distintos dominios.

## 4. Epoch Database – Notable Models
- **Origen:** Epoch AI
- **Descripción:** Base de datos pública de modelos de IA notables, con información como fecha de publicación, organización y dominio. Se utilizó para analizar la evolución temporal de modelos de lenguaje y la participación de organizaciones desarrolladoras en el streamgraph.
